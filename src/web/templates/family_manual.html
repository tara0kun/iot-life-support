第Ⅰ部 ─ 説明書

1. はじめに ─ なぜこのシステムを作ったか

解決したい問題

認知症のおばあちゃんが抱えている3つの困りごと：

  1. 🍚 食事を何度も摂ってしまう（最優先 / Phase 1で対応）
    食べたことを忘れて何度も食事を準備してしまう。家族が見ていない時の繰り返しが懸念。
  2. 🚪 訪問販売・勧誘への対応（Phase 2予定）
  3. 🚿 入浴時に頭を洗わない（Phase 2予定）

設計の4つの原則

原則 意図
気づかせない 「監視されている」と感じさせない。タブレットは「自分の記録帳」として説明
命令しない 「食べちゃダメ」と直接指摘せず、機械の不調に見せて物理的に阻止する
多層防御 1段階目：そっと気づかせる → 2段階目：物理阻止 → 3段階目：家族通知
家族操作は見えない 編集・バイパス・ロック解除はおばあちゃんに絶対知られない仕組み

家族での合意事項（2026-04-17）

項目合意内容
Phase 1のスコープ食事問題のみに絞る
祖母への説明「記録帳」として伝える（秘密ではない）
祖父の同意取得済み
運用担当孫（システム）＋ 母（日々の対応）
投入時期2026年GW中
中断基準記録しなくなる／反発／機器破壊／ずるをし始めた

2. システムの仕組み

🏠 設置している機器

機器役割設置場所
Tapo P110M（スマートプラグ）炊飯器の電力監視＋電源ON/OFF制御炊飯器のコンセント
Tapo C220（カメラ）顔認識で誰がキッチンにいるか判別壁時計の横（高い位置）
Tapo H100（ハブ）T110/T100センサーをWi-Fiに中継ルーター付近
Tapo T110（開閉センサー） ×3冷蔵庫ドア・浴室ドア の開閉検知冷蔵庫／浴室ドア
Tapo T100（モーション）脱衣所での動きを検知（入浴中の安全確認）脱衣所
Raspberry Pi 5すべてを統合する小型コンピュータルーター付近
Androidタブレットおばあちゃん用の記録帳画面キッチンテーブル

🔄 データの流れ（食事検知の例）

  1. 炊飯器の電源が入る → P110Mが 1100W を検知
  2. カメラが 祖母の顔 を認識（顔認識DB照合）
  3. 冷蔵庫ドアが開閉（T110が検知）
  4. 3つのセンサー反応を 15分以内のセッション として集約
  5. 「朝食」セッションとしてDB記録
  6. タブレットのスタンプに「✓」表示／お花が成長

🛡️ 食べ過ぎ防止の仕組み

  1. 1回目の食事を検知 → DBに記録
  2. その後 90分以内 に冷蔵庫開閉＋カメラ＋電力 のいずれかを検知
  3. 「2回目の食事行動」と判定 → 炊飯器の電源を 自動でOFF（Matter経由）
  4. 家族にLINE通知「🍚 祖母が食事行動を検知しました」
  5. 2時間後に自動で電源復帰（または家族が手動解除）

おばあちゃんには「炊飯器が壊れた」と見える → 反発が起きにくい

🌐 通信経路

通信方式
P110M ⇄ ラズパイMatter（python-matter-server経由）
T110/T100 ⇄ ラズパイH100ハブ → Wi-Fi → python-kasa
C220 ⇄ ラズパイRTSP（ストリーミング）
ラズパイ ⇄ 家族のスマホCloudflare Tunnel経由のHTTPS
ラズパイ ⇄ LINELINE Messaging API（双方向）

💾 データの保存

  • SQLite（ラズパイ内）に全データを保存
  • 毎日3時に自動バックアップ（14日分保持）
  • 顔データはラズパイのローカルにのみ保存（クラウドに送らない）
  • カメラ映像は人物検知時に静止画のみ保存（動画録画はしない）

4. プライバシーと倫理について

守っていること

  • 映像はローカル保存のみ：クラウドや外部サービスに送信しません
  • 顔データもローカルのみ：祖母・家族の顔特徴量はラズパイ内で完結
  • 動画録画なし：人物検知時の静止画のみ（14日で自動削除）
  • 家族のみアクセス：パスワード認証＋トークン認証
  • 監査ログ：家族の編集・解除はすべて記録（後から追跡可能）

悩ましい点・継続して考えること

  • 「監視されている」と本人が気づいた時の対応 → 中断基準を明確化済み
  • 家族間の意見対立時の決定権 → 母を中心に運用
  • 事故時の責任所在 → 物理的な事故防止より「気づき」の補助としての位置づけ
  • 祖母の自尊心への配慮 → 「できた」を強調するUI設計

🔓 ロック解除の手順（LINE）

1
「ロック解除」とLINEに送信
2
4桁の確認コードが返ってくる（例: 3847）
3
5分以内にその4桁を返信
4
「✅ 解除しました」と返信が来たら完了
注意：3回間違えると15分間ロックされます。確認コードは5分で無効になります。

7. 家族管理画面の使い方

▲ 家族管理画面（スマホ表示）

🔝 ヘッダー

  • 週次レポートボタン：過去7日のまとめを表示（PDF保存可）
  • ログアウトボタン

📊 ステータスカード（4つの数字）

  • 今日のイベント：センサーが検知した数
  • 登録人数：システムに登録された家族の人数
  • 消費電力 (W)：炊飯器のリアルタイム電力
  • 祖母の食事回数：3回以上で赤く警告

🔒 機器管理セクション

炊飯器のロック状態が常に見えます。

  • 🔓 通常：使える状態。「手動ロック」ボタンで予防的にロック可能
  • 🔒 ロック中：自動ロックされている状態。「ロック解除」で解除可能

📊 7日間アクティビティ（ヒートマップ）

過去7日のセンサー反応を時間帯別に可視化。色が濃いほど活動が多いです。

  • 食事数・お風呂数・トイレ数のサマリーが上に表示されます
  • マスをタップすると件数が見えます

📝 かんたん記録

おばあちゃんから「薬飲んだよ」など報告を受けたら、対応するボタンを押して記録します。

  • 起床 / お薬 / お風呂 / トイレ / おやつ / 就寝 の6種類
  • 誤タップ防止の確認ダイアログ付き
  • 記録すると即座にタブレットのスタンプにも反映

📋 家族タスク（役割分担）

「朝のお薬確認は母」のように担当を決めてリマインドできます。

項目意味
タスク名例：朝のお薬確認、夜の様子見
担当例：母、孫
時刻0〜23（その時刻にLINEで担当者に通知）
状態⬜ 未対応 / ✅ 完了

完了したらLINEで「済 タスク名」と送信、または家族画面で操作してください。

💊 お薬スケジュール

朝・昼・夜のそれぞれで「何時に飲むか」を設定。

  • その時刻にタブレットにリマインドが表示される
  • 未設定なら表示されない（飲まない時間帯はそのまま）

🍚 炊飯量ガイド

家族が「次に何合炊くか」を設定すると、おばあちゃんのタブレットに表示されます。

  • 1合 / 2合 / 3合のプリセット or 自由入力
  • クリアすると非表示になります

💬 タブレットに伝言

おばあちゃんへのメッセージをタブレットに表示できます。

  • プリセット（お薬／お風呂／ご飯／電話）または自由入力
  • 表示時間は60分（自動で消える）
  • おばあちゃんが「わかった」を押すと消える

⚙️ 詳細設定（折りたたみ式）

普段は触らなくてOK。以下を変更できます：

  • 各種通知のON/OFF（医薬・お風呂・まとめ・タスク・異常検知・週次・URL）
  • 異常検知のしきい値（無反応時間・深夜時間帯・冷蔵庫開放時間）
  • 食事ロックの判定窓・食べ過ぎ警告のしきい値

📅 イベント一覧

▲ 全イベントの履歴（人物・ソース別）
  • 日付ナビで前日・翌日に移動可能
  • 人物フィルタ（全員 / 祖母 / 母 等）
  • 各イベントの時刻クリックで時刻編集
  • 各イベントの人物ドロップダウンで人物変更
  • 右端の×ボタンで削除（確認ダイアログあり）

📊 週次レポート（ヘッダーから）

過去7日のまとめがブラウザで表示されます。

  • 食事回数・スタンプ達成率・お薬服用率・お風呂日数・自動ロック発動回数
  • 「印刷／PDF保存」ボタンでPDFに変換可能
  • ケアマネージャーや医師への共有に使えます

8. おばあちゃんのタブレット画面

おばあちゃんが見る画面です。家族が直接操作することは基本的にありません。

▲ 時計＋時間帯ガイド＋注意喚起

主な構成

  • 時計＋日付＋挨拶：「おはようございます」など時間帯に応じた挨拶
  • 家族からの伝言：青いカードで表示。「わかった」ボタンで消える
  • 注意喚起：「さっき食べましたよ」「お薬飲みましたか？」など
  • 最後に食べたのは：直近の食事ラベルと時刻
  • お花：1日のスタンプ達成数で成長（種→芽→花→満開）
  • スタンプカード：今日できたことが「✓」で記録される
  • 「できた」ボタン：起床・お薬・お風呂・就寝で表示
  • わたしのおにわ：過去14日のお花が並ぶ
  • 音声ON/OFFトグル：ヘッダー右、家族メッセージなどを読み上げ
▲ お花の成長とスタンプカード
▲ わたしのおにわ（過去14日のお花）

9. 困ったとき

📱 タブレットが真っ白になった

  • Wi-Fiが切れている可能性 → 黄色いバナー「接続が切れています」が出ているか確認
  • 復活すると自動で再読み込みされます
  • それでもダメなら：タブレットの電源を入れ直す

🌐 LINEで届いたURLが繋がらない

  • LINEで「リンク」と送ると最新URLが返ってきます
  • 古いURLは無効になっています

🔒 炊飯器が動かない（ロック中）

  • 家族管理画面の「機器管理」セクションで「ロック解除」ボタン
  • もしくはLINEで「ロック解除」→ 4桁コード返信

⚠️ 通知が多すぎる/少なすぎる

  • 家族管理画面の「⚙️ 詳細設定」を開く
  • 各通知のON/OFF、しきい値（4時間→6時間など）を変更

🚨 「センサー無反応」アラートが届いた

おばあちゃんに電話／訪問して安否を確認してください。 4時間以上センサーが反応していません。

🚨 「深夜の家電稼働」アラートが届いた

深夜2時〜5時に炊飯器が動きました。 徘徊や認知症による異常行動の可能性があります。至急確認してください。
