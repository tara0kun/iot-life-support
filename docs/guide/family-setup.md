# 家族用 セットアップ手順

> このガイドは、新しい家族メンバーが LINE ボットと家族管理画面を使えるようになるまでの手順です。
> 所要時間: 約10分。

---

## ステップ1: LINE で友達追加

1. 家族グループ LINE の固定メッセージにある **「ばあちゃんIoTサポート」の友達追加リンク** をタップ
2. 「友達追加」を押す
3. トークルームを開く

---

## ステップ2: 自分を登録する

LINE トークで **「登録 (自分の名前)」** と送信してください。

例:
```
登録 ゆきこ
登録 みきこ
登録 はるか
登録 おじいちゃん
```

> ✅ **確認**: 「✅ 〇〇さんを家族登録しました」と返信が来れば成功
> 同じ名前を再送すれば情報を更新できます

### 名前について

- 自由な名前でOK（呼び名でも本名でも）
- 後から変えたい時は **「登録 新しい名前」** で再登録すれば上書き
- 削除したい時は **「登録解除」** とだけ送信

---

## ステップ3: 家族管理画面（家族UI）にアクセス

家族 LINE グループの固定メッセージにある **家族管理画面の URL** をブラウザで開きます。

### URL の例
```
https://〇〇〇.trycloudflare.com/family
```

### パスワード

家族 LINE グループの固定メッセージに記載されています（通常 `1488` など簡易な4桁数字）。

> ⚠️ パスワードは **家族間でのみ共有** してください。

### ログインできた？

- 入れない時 → URL が変わっている可能性あり。LINE で **「リンク」** とだけ送ると最新URLが返ってきます

---

## ステップ4: 主要画面の使い方

### 「今日の行動」セクション
祖母さんの今日の動きが時系列で表示されます。

### 「かんたん記録」セクション
祖母から「お薬飲んだ」と聞いた時など、その場でボタンで記録できます。

| ボタン | いつ使う |
|---|---|
| 🌅 起床 | 起きた時間を聞いた時 |
| 💊 お薬 | お薬を飲んだことを確認した時 |
| 🛁 お風呂 | お風呂に入ったと聞いた時 |
| 🚽 トイレ | （通常はセンサーが自動記録） |
| 🍪 おやつ | おやつを食べた時 |
| 🌙 就寝 | 寝た時間を確認した時 |
| 🍴 外食 | 一緒に外食した／外で食べたと聞いた時（朝/昼/夕/間食を選択） |
| 🌅 朝食 / 🍱 昼食 / 🍚 夕食 / 🍰 間食 | センサーが見逃した時の手動記録 |

押すと家族全員にLINEで「📝 家族が『〇〇』を記録しました」と通知されます。

### 「タブレットに伝える」セクション
祖母のタブレットにメッセージを送れます（即時表示+音声）。

### 「機器管理」セクション
炊飯器の手動ロック / ロック解除ができます。

### 「7日間の活動記録」セクション
食事/お風呂/トイレ/冷蔵庫/脱衣所 のタブで切り替えてヒートマップ表示。

### 「ライブ映像（リビング C220）」セクション
必要な時だけ ON でリビングのカメラ映像を見られます。

## ステップ5: 通知の対応方法

### 通知が来たら

LINE で通知が来たら **適切なボタンを押す** ことで対応します:

| ボタン | 意味 |
|---|---|
| ✓ 了解（対応します） | 内容を理解した、対応する |
| 対応不要（誤検知） | これはセンサーの誤検知、今後再通知しないで |

### 食事系の通知

「🍱 〇〇頃 「昼食」の動きをセンサーが検知しました」が来たら:
- [祖母さん] / [祖父さん] / [母さん] / [他の家族] / [食事じゃない] から選ぶ

### ロック確認

「🍚 祖母さんが本日3回目の食事を検知しました」が来たら:
- [ロックする] → 炊飯器をロック
- [ロックしない] → ロックスキップ

### 誰かが対応すると全員に共有

家族の誰かが対応すると **「☑️ 〇〇さんが対応しました」** がグループに流れます。
これで家族間で重複対応を避けられます。

## よくある質問

### Q1. 通知が来すぎる
→ 「対応不要」を押し続けると、システムが「これは誤検知」と学習して減ります。
それでも多いと感じたら開発者（孫）に連絡。

### Q2. 自分宛じゃないと思う通知が来る
→ 全員に同じ通知が届きます。誰か対応すれば全員の通知が完了マークされます。

### Q3. 家族 UI のリンクが切れた
→ LINE で **「リンク」** と送ると最新 URL が来ます。
