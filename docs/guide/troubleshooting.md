# トラブルシューティング

> 困った時の FAQ。ほとんどは **再起動** で直ります。

---

## タブレット関係

### Q. タブレットの画面が暗い
**A.** 画面をタッチ → 明るくなる。長時間放置していたかもしれません。

### Q. アプリが固まった
**A.** 以下の順で試してください:
1. 画面ピン留めを解除（戻る + タスクボタン同時長押し → PIN）
2. アプリを終了 → ホーム画面のアイコンから再起動 → 再度ピン留め
3. それでもダメなら **タブレット再起動**（電源ボタン長押し → 再起動）

### Q. 画面ピン留めが解除できない
**A.** 機種によって解除方法が違います:
- 戻るボタン + タスクボタン **同時長押し**
- ボリュームダウン + 電源 **同時長押し**
- ホームボタン **長押し**

それでもダメなら強制再起動（電源ボタン10秒長押し）。

### Q. ホーム画面のアイコンが消えた
**A.** ブラウザで URL を開き直して **「ホーム画面に追加」** をやり直してください。手順は [tablet-setup.md](tablet-setup.md) のステップ2。

### Q. 写真が撮れない
**A.** カメラ権限を確認:
1. 設定 → アプリ → Chrome（または使ってるブラウザ）
2. 権限 → カメラ → **「許可」** に設定

### Q. 音声リマインドが鳴らない
**A.**
1. 設定 → 音 → メディア音量を上げる
2. ブラウザ画面で1度どこかをタップ（ブラウザの自動音声ポリシーで初回タップが必要）

## LINE 関係

### Q. LINE 通知が来ない
**A.**
1. LINE で `タブレットIoTサポート` のトークを開いて「リンク」と送る → 最新URLが返るか確認
2. 返信が来ない場合 → サーバー側の問題。孫に連絡

### Q. 「登録 〇〇」が反応しない
**A.**
1. ボットを **友達追加** したか確認（ブロックしていないか）
2. もう一度送ってみる
3. 返答が「未登録」のままなら、ボットURLを再確認

### Q. 通知が多すぎる
**A.** 「対応不要（誤検知）」ボタンを押し続けると、システムが学習して減ります。
それでもストレスなら、孫に「夜間通知抑制」を設定してもらう。

### Q. 通知のボタンを押し間違えた
**A.** 残念ながら一度押すと取り消せません（後勝ちで上書きはされる）。家族管理画面で履歴確認できます。

## 家族管理画面 関係

### Q. URL を開いてもエラー
**A.**
1. LINE で「リンク」と送る → 最新URL取得
2. パスワード（家族 LINE グループの固定メッセージ参照）を確認
3. それでもダメなら 5分後に再試行（Cloudflare Tunnel が再起動中の可能性）

### Q. ライブ映像が表示されない
**A.**
1. 「▶ 映像ON」をもう一度押す
2. ブラウザのキャッシュクリア → リロード
3. それでもダメなら、カメラの電源とWi-Fi接続を確認

### Q. ボタンを押しても反応がない
**A.**
1. ページをリロード（更新ボタン）
2. ログアウトしてログインし直す（パスワード再入力）

## センサー関係

### Q. 〇〇が反応しない
**A.** **電池切れの可能性が一番高い**:
- T110/T100 → 電池が CR2 か CR2032。Tapo アプリで電池残量確認
- SwitchBot → 単4電池。長押しで残量確認

電池交換手順は機器の取扱説明書参照。

### Q. 全センサーが動かない
**A.** **H100 ハブの電源** を確認:
1. ハブの LED が点いているか
2. Wi-Fi につながっているか（Tapo アプリで確認）
3. ダメなら **ハブを抜き挿し**（リセット）

### Q. カメラに「未識別」と出る
**A.** カメラの位置を変えたか、その人がまだ顔登録されていません。孫に「〇〇さんを顔登録して」と頼む。

## システム全般

### Q. なんか変な動きをする
**A.** **再起動** が魔法の薬:
1. ラズパイ再起動: `ssh taraberrypi` で接続 → `sudo reboot`
2. 5分待つ
3. ほぼ全部直ります

### Q. ラズパイが応答しない
**A.**
1. **電源を抜き挿し**（最終手段）
2. それでもダメ → 孫に連絡（祖母宅に行く必要あり）

### Q. 開発者に質問・要望を送りたい
**A.** （LINE リッチメニュー実装後）「意見を送る」ボタンから送信、または LINE で「意見」とだけ送ると質問入力モードになる予定。

- 困った時は **家族 LINE グループ** に投稿
- 緊急時は孫に直接電話
- システムに関する質問は LINE で「意見」と送信（実装予定）

## 障害が起きた時のチェックリスト

孫が祖母宅に行けない時、家族で確認できる項目:

1. [ ] **ラズパイ** の電源 LED 点灯（赤・緑）
2. [ ] **ラズパイ** の Wi-Fi LED 点滅
3. [ ] **H100 ハブ** の LED 点灯（白）
4. [ ] **C220 カメラ** の LED 点灯（緑）
5. [ ] 各 Tapo 機器が **Tapo アプリ** に表示される
6. [ ] **タブレット** が起動して画面が見える
7. [ ] **インターネット** が繋がる（他のスマホで Wi-Fi 確認）

これら全部 OK なのに動かない時は、サーバーソフトウェアの問題。孫に連絡。
