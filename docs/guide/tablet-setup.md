# 祖母タブレット セットアップ手順

> このガイドは、家族が祖母用タブレットを **誰でも安全にセットアップできる** ように手順を丁寧に書いてあります。
> 上から順番にやれば終わります。所要時間: 約30分。

## 用意するもの

- Android タブレット（10インチ前後推奨、Wi-Fi接続済みのもの）
- タブレットの 解除パスワード or PIN（4桁以上、後で何度か使う）
- 祖母宅の Wi-Fi に接続済みであること
- システムの URL（家族グループ LINE の固定メッセージにあります）
- 設置場所の確認（祖母がいつもいる場所、コンセント近く）

---

## ステップ1: 充電と Wi-Fi

1. タブレットの電源を入れて、充電ケーブルを挿す
2. 設定 → ネットワーク → Wi-Fi → **祖母宅の Wi-Fi** に接続
3. 充電 100% になるまで使い始めない（電池が劣化しにくい）

---

## ステップ2: ブラウザで URL を開く

1. **Chrome** または **Edge** を起動
2. アドレス欄に URL を入力（家族 LINE の固定メッセージ参照）
3. ページが表示されたら **「📱 ホーム画面に追加」** を押す
   - Chrome: 右上「⋮」メニュー → 「ホーム画面に追加」
   - Edge: 右下「…」 → 「アプリにインストール」

これでホーム画面にアプリのアイコンができ、タップだけで起動できるようになります。

> ✅ **確認**: ホーム画面にアプリアイコンが表示されていること

---

## ステップ3: 画面ピン留め（重要）

> 祖母が誤タッチで他のアプリに移ってしまうのを防ぎます。**この設定は必須です。**

### 設定の有効化

1. **設定** アプリを開く
2. **セキュリティ** または **パスワードとセキュリティ** をタップ
3. **詳細設定** → **アプリの固定** （または「画面の固定」「Screen Pinning」）
4. スイッチを **オン** にする
5. 「**固定解除前にPINを要求**」も **オン** にする

> 💡 メーカーや機種によって項目名が違います:
> - **Galaxy（Samsung）**: 設定 → セキュリティとプライバシー → その他のセキュリティ設定 → アプリの固定
> - **Xperia（SONY）**: 設定 → セキュリティ → アプリの固定
> - **Pixel**: 設定 → セキュリティ → 詳細設定 → アプリの固定
> - **Lenovo / HUAWEI / OPPO**: 設定 → セキュリティ → アプリピン留め
> - 見つからない場合は「**画面ピン留め**」または「**screen pinning**」で設定アプリ内検索

### アプリの固定（祖母に渡す前に毎回行う）

1. ホーム画面の **アプリアイコンをタップして起動**
2. **タスク画面**を開く（画面下からスワイプアップ→止める、または□ボタン）
3. アプリのプレビュー画面の **アイコンを長押し** → **「固定」** または **「ピン留め」**
4. 確認画面で **「OK」** を押す

これで他のアプリには切り替えられなくなります。

### 解除方法（家族のみ知ってればOK）

- 画面下のナビゲーションバーで **「戻る」+「タスク」を同時に長押し**（端末により違う場合あり）
- PIN を入力して解除

> ⚠️ 祖母には解除方法を**教えないでください**。誤って解除して他のアプリを開いてしまう原因になります。

---

## ステップ4: スリープ防止

タブレットが暗くなると祖母が困るので、画面を常時点灯させます。

1. **設定** → **ディスプレイ** → **スリープ** または **画面消灯時間**
2. **「なし」** または **最長**（30分など）を選択
3. （推奨）**設定** → **電池** → **充電中は画面オン** をオン

> 💡 ブラウザのアプリ自体が WakeLock API でスリープを防止しますが、Android の設定で念のため最長にしておきます。

---

## ステップ5: 通知 OFF（誤操作防止）

タブレットに余計な通知が出ると祖母が混乱します。すべて切ります。

1. **設定** → **通知** → **アプリ通知**
2. すべてのアプリで通知を **オフ** にする（または通知アクセス全体をオフ）
3. **設定** → **音と振動** → **着信音量・通知音量** を **0** にする
4. **設定** → **マナーモード** を有効化（バイブも切る）

---

## ステップ6: 物理ボタンの工夫

タブレットの **音量ボタン** や **電源ボタン** を祖母が間違って押す可能性があります。

- **電源ボタン**: 軽くカバーするか、押しにくい向きに置く
- **音量ボタン**: 同上
- できれば **画面が常に上向き** になる スタンドを用意

---

## ステップ7: ブラウザ設定（誤操作対策）

### Chrome の場合

1. Chrome を開く → 右上「⋮」 → **設定**
2. **オートコンプリートとパスワード** → 全部 **オフ**
3. **支払い方法** → **オフ**
4. **広告** → **オフ**
5. **お知らせ** → **オフ**

### URL バーを隠す（任意）

PWA としてホーム画面から開くと URL バーは表示されません。**必ずホーム画面のアイコンから開く**ようにしてください。

---

## ステップ8: 動作確認

セットアップが終わったら以下を確認:

- [ ] ホーム画面のアイコンをタップ → アプリが起動する
- [ ] 画面が暗くならない（5分待って画面ON）
- [ ] 「できた」ボタンを押すと確認モーダルが出る
- [ ] 「いいえ、まだです」を押すと記録されない
- [ ] 「はい、できました」を押すと記録される
- [ ] 戻るボタンを押しても画面ピン留めで止まる
- [ ] 家族から「タブレットに伝える」を送ると、勝手に画面が更新される + 音声が流れる

すべてOKなら祖母に渡しても大丈夫です。

---

## ステップ9: 祖母への引き渡し

祖母には以下だけ伝えれば大丈夫です（[grandma-usage.md](grandma-usage.md) も渡してください）:

1. 「**画面の指示**に従って動いてね」
2. 「**できた**ボタンを押すと、家族に教えてくれるよ」
3. 「**わからない時、家族に電話して**ね」
4. **解除方法は教えない**（誤操作防止）
