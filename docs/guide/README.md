# ばあちゃんIoTサポート 使い方ガイド

このフォルダには、家族が誰でも IoT システムを設定・使えるようにするためのガイドが入っています。

## 📖 最初に読むもの

**スマホ・パソコンが苦手な家族**は、まず以下を読んでください:

- 🌟 [**毎日の使い方ガイド**](daily-usage.md) — シニア家族向け、1日数回の簡単な操作を説明
  - 「LINE で通知が来たらボタンを押すだけ」というシンプル版
