# ばあちゃんIoTサポート 使い方ガイド

このフォルダには、家族が誰でも IoT システムを設定・使えるようにするためのガイドが入っています。

## ガイド一覧

| ガイド | 対象 | 内容 |
|---|---|---|
| [tablet-setup.md](tablet-setup.md) | 設定する人（家族） | 祖母タブレットの初期設定（一番重要） |
| [grandma-usage.md](grandma-usage.md) | 祖母 | タブレットの使い方（イラスト多め） |

| [troubleshooting.md](troubleshooting.md) | 全員 | トラブル時のFAQ |

## こんな時はこのガイド

- 「祖母にタブレットを渡す前に何を設定すればいい？」 → [tablet-setup.md](tablet-setup.md)
- 「LINE で通知が来た、どうすればいい？」 → [family-reference.md](family-reference.md)
- 「家族 LINE グループに新しい人を追加したい」 → [family-setup.md](family-setup.md)
- 「動かない！」 → [troubleshooting.md](troubleshooting.md)
