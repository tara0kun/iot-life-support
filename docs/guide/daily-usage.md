# 毎日の使い方ガイド（家族向け）

> このガイドは「**スマホ・パソコンに詳しくなくても使える**」を目指して書かれています。
> 最初に1回だけ読めば、毎日の操作で迷わなくなります。

---

## このシステムでできること

おばあちゃんの家にあるセンサーやカメラが、こんなことを教えてくれます:

- 🍱 ごはんを食べたか
- 🛁 お風呂に入ったか
- 🚽 トイレに長くいないか（心配な時だけ）
- 🔒 炊飯器を何度も使いすぎていないか
- 📷 写真でおばあちゃんの食事を共有

そして、何かあると **LINE** や **家族専用ページ** で家族にお知らせします。

---

## 毎日やってほしいこと（1日数回）

### 🌟 やることは1つだけ

**LINE で通知が来たら、ボタンを押す**。これだけです。

例えばこんな通知が来ます:

```
🍱 12:40頃 「昼食」の動きをセンサーが検知しました
誰の食事ですか？
[祖母さん] [祖父さん] [母さん] [他の家族] [食事じゃない]
```

→ ボタンの中から **当てはまるもの** をタップしてください。
→ 分からなければ **「他の家族」** か、明らかに違うなら **「食事じゃない」** を押してください。

> 💡 1人がボタンを押すと、他の家族全員に「○○さんが対応しました」と共有されます。**みんなで重複して対応しなくてOK**。

---

## LINE の通知を見逃した時

LINE の通知をうっかり見逃しても大丈夫です。

### 家族専用ページの「📨 未対応の確認」へ

1. LINE で **「メニュー」** と送る
2. 返信のボタンから **「🏠 家族管理画面」** を押す
3. パスワード（家族 LINE グループの固定メッセージに記載）を入力
4. ページの上の方にある **「📨 未対応の確認」** という欄を見る

ここに **対応されていない通知** が全部溜まっています。

```
[📅 12:40]  🍱 食事候補の確認
12:40頃 「昼食」の動きをセンサーが検知しました
判定根拠: 炊飯器の蓋×2、冷蔵庫×3
誰の食事ですか？
[祖母さん] [祖父さん] [母さん] [他の家族] [食事じゃない]
```

**オレンジ色の「📅 12:40」が出来事の起きた時刻** です（LINE が届いた時刻ではありません）。

→ ボタンを押して対応すれば、その通知は消えます。

> 💡 24時間放っておくと自動的に消えます（タイムアウト扱い）。

---

## よくある通知と対応の仕方

| 出てくる通知 | やること |
|---|---|
| 🍱 食事候補 | **誰が食べたか** を選ぶ。分からなければ「他の家族」 |
| 🛁 浴室で湿度上昇 | **誰が入浴したか** を選ぶ。湿度だけ上がって誰もいない時は「清掃」or「湯はり中」 |
| 🔒 炊飯器をロックしますか？ | 何度も食べすぎ防止ならロック。気にしなければ「ロックしない」 |
| ⚠️ トイレに 5分以上 | **電話してみる**。返事があれば安心。なければ家族の誰かが見に行く |
| 🚨 浴室で30分動きなし | **すぐ電話 or 訪問**。緊急の可能性 |
| 💇 髪洗い忘れ | 声かけしてあげる |
| 📝 家族が記録 | 情報のみ、対応不要 |
| ☑️ 〇〇さんが対応 | 情報のみ、誰かが対応した報告 |

---

## 「これは違うな」と思ったら

センサーは時々間違えます。例えば:
- 「食事候補」と言っているけど、実は炊飯器の蓋を確認しただけ
- 「浴室で湿度上昇」と言っているけど、実は祖父が顔を洗っただけ

そんな時は **「食事じゃない」** や **「誰もいない」** ボタンを押してください。
**システムが学習して、次は同じ間違いをしにくくなります。**

---

## 食事の手動記録

おばあちゃんが食事したのにセンサーが気づかなかった時、または **外食** した時は家族側で記録できます:

1. 家族管理画面の **「かんたん記録」** セクション
2. ボタンを押す:
   - 🍴 外食 → 外食の朝/昼/夕を選択
   - 🌅 朝食 / 🍱 昼食 / 🍚 夕食 / 🍰 間食 → センサー見逃し時の手動記録
3. 記録すると家族全員に「📝 家族が『〇〇』を記録しました」と共有されます

---

## おばあちゃんへのメッセージ送信

タブレットを通じておばあちゃんに伝言できます:

1. 家族管理画面の **「タブレットに伝える」** セクション
2. メッセージを入力
3. 優先度を選ぶ:
   - **緊急**: 大きな音と繰り返し読み上げ
   - **通常**: 普通の音声で伝える
   - **静か**: 表示だけ、音声なし（夜中等）
4. 送信ボタン

→ **即座にタブレットに表示** + **音声で読み上げ** されます。

---

## 困った時の優先順位

1. **どうしても分からない時**: LINE で **「ヘルプ」** と送る → コマンド一覧が返ってくる
2. **意見や質問**: LINE で **「意見 〇〇〇〇」** と送る → 開発者に届く
3. **緊急時**: 直接電話してください

---

## 「これだけ覚えとけば OK」3つのコマンド

LINE で送ると、すぐに役立つコマンド3つ:

| コマンド | 結果 |
|---|---|
| `メニュー` | 主要機能へのボタンが表示される |
| `リンク` | 家族管理画面の最新URLが返ってくる（URL変わった時用） |
| `ヘルプ` | 全コマンド一覧 |

## 印刷して家族で共有する

このページを印刷して冷蔵庫に貼ったり、家族グループ LINE に共有すると便利です:

1. ブラウザでこのページを開く
2. 右下の **「🖨️ 印刷 / PDF」** ボタンを押す
3. 「PDFに保存」または「印刷」を選択
