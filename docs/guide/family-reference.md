## 通知の種類

### 🍱 食事関係

#### 「〇〇頃 「昼食」の動きをセンサーが検知しました」
**意味**: 炊飯器・冷蔵庫・脱衣所モーションなどの動きから「食事の可能性」を検知。
**選択肢**:
- [祖母さん] / [祖父さん] / [母さん] / [他の家族] → その人の食事として記録
- [食事じゃない] → 誤検知扱い、再通知なし

#### 「🍚 祖母さんが本日3回目の食事を検知しました」
**意味**: 祖母さんが今日3回目以上の食事を検知。多すぎる可能性。
**対応**: 様子を見るか、祖母さんに連絡。

#### 「🍴 祖母が食事をしてからXX分経過」
**意味**: 食事から指定時間経過した状態。日中の長時間絶食を検知。

### 🛁 お風呂関係

#### 「🛁 浴室で湿度上昇を検知」
**意味**: 湿度・温度・ドア状態・モーションから入浴の可能性を検知。
**選択肢**:
- [祖母] / [祖父] / [母] / [他] → その人の入浴として記録
- [湯はり中] → 人不在、湯はりだけ
- [清掃] → 浴室掃除中
- [誰もいない] → 誤検知

#### 「🚨 浴室で30分以上動きなし」
**意味**: お風呂に入ってから30分動きがない。**緊急確認** が必要。
**対応**: 即座に祖母に連絡、必要なら祖母宅に直行。

#### 「💇 祖母がお風呂後にドライヤーを使った形跡がありません」
**意味**: ドライヤーセンサー稼働時のみ。髪が濡れたまま放置の可能性。
**対応**: 髪を乾かすよう声かけ。

### 🔒 機器ロック関係

#### 「🍚 炊飯器をロックしますか？」
**意味**: 食事2回以上検知時、炊飯器ロックの確認。
**選択肢**:
- [ロックする] → 炊飯器をロック
- [ロックしない] → スキップ

#### 「🔒 炊飯器を手動ロックしました」
**意味**: 家族の誰かが手動でロックした。情報通知のみ。

#### 「🔓 炊飯器のロックを家族が手動で解除しました」
**意味**: 家族の誰かが手動で解除した。情報通知のみ。

### 🔁 再通知

#### 「🔁 再通知（2回目）」
**意味**: 30分応答がなかった通知の再送。最大3回まで。

#### 「⏰ 90分以上応答なし」
**意味**: 3回の通知に応答なし、タイムアウトで諦めた状態。
**対応**: 後で家族管理画面で確認。

### 📝 記録・操作

#### 「📝 家族が『〇〇』を記録しました」
**意味**: 家族の誰かがかんたん記録ボタンを押した。情報通知。

#### 「☑️ 〇〇さんが対応しました」
**意味**: 通知に対して家族の誰かが対応完了。重複対応を避けるための共有。

## 行動サマリの読み方

家族UIの「今日の行動」セクションで表示される項目:

| アイコン | 意味 |
|---|---|
| 🚽 トイレ使用 | 開閉から滞在時間まで表示 |
| 🧊 冷蔵庫使用 | 短時間の開閉を集約 |
| 🍚 炊飯器稼働 | 電力推移と継続時間 |
| 🛁 浴室に入った / 出た | ドア閉/開のタイミング |
| 🚿 シャワー使用検知 | 湿度急上昇から推測 |
| 🚶 脱衣所で動き | モーションのクラスタ |
| 📝 家族が記録 | 手動記録 |

> 💡 名前（祖母さん）が表示されるのは **カメラで顔認識した時** や **家族が手動で誰か指定した時** だけです。
> センサー単独で「祖母さん」と推測している場合は名前を表示しません（誤帰属を避けるため）。

## 用語

| 用語 | 意味 |
|---|---|
| セッション | 数分間で連続して起こった行動のまとまり |
| 確定（confirmed） | 家族が LINE で「これは〇〇」と確認したセッション |
| 未確定 | 家族確認待ちのセッション |
| 誤検知 | センサーは反応したが実際は違う行動だったもの |
| アクティブパーソン | カメラの直近識別から推測される、その時いた人 |
| ベースライン湿度 | 普段の状態の湿度（通常 50-60%程度） |

## センサーの仕組み（簡単版）

### 食事の検知方法
1. 炊飯器の蓋を開ける
2. 冷蔵庫を開ける
3. 食卓近くで動く
4. これらが組み合わさると「食事してそう」と推測 → LINE で確認

### お風呂の検知方法
1. 風呂のドアを閉める
2. 脱衣所で動きが続く
3. 浴室の湿度が上がる
4. これらが組み合わさると「入浴してそう」と推測 → LINE で確認

### トイレの検知方法
1. トイレのドアを開閉する
2. ドア開閉のセットで「トイレ使用」として記録
（誰がというのはカメラと組み合わせて推測）
