# LINE操作ガイド

> ばあちゃんIoTサポートをLINEから操作するための完全ガイド。
> 印刷したい場合は、画面右下の「🖨️ 印刷 / PDF」ボタンから「PDFに保存」を選んでください。

---

## 1. 最初にやること（友達追加と自己登録）

### 1-1. LINE ボットを友達追加

家族 LINE グループの固定メッセージにある **「ばあちゃんIoTサポート」の友達追加リンク** をタップして友達追加します。

### 1-2. 自分を登録する

トーク画面で以下のように送信:

```
登録 ゆきこ
```

「✅ 〇〇さんを家族登録しました」と返信が来れば成功です。

**ポイント**:
- 名前は呼び名でも本名でもOK
- 後から変更したい場合は「登録 新しい名前」で上書き
- やめたい時は「登録解除」と送信

---

## 2. メニューを呼び出す

LINE のトーク画面で以下を送信:

```
メニュー
```

すると下部にボタンが並びます:

| ボタン | 動作 |
|---|---|
| 🏠 家族管理画面 | 家族用のWeb画面を開く |
| 📚 使い方ガイド | このガイド一覧を開く |
| 📷 顔学習 | 未識別の顔をラベル付け |
| 📷 食事写真 | 祖母の食事写真ギャラリー |
| 💬 意見・質問 | 開発者への意見送信モード |
| 🆘 困った時 | トラブル時のFAQ |

タップすると対応するページが開きます（家族管理画面はパスワード認証あり）。

---

## 3. 通知への対応

### 3-1. 通知の種類

| 種類 | 内容 | 対応の必要性 |
|---|---|---|
| 🍱 食事候補 | 「12:30頃 昼食の動きを検知しました」 | 誰の食事か選択 |
| 🛁 お風呂候補 | 「浴室で湿度上昇を検知」 | 誰の入浴か選択 |
| 🍚 ロック確認 | 「炊飯器をロックしますか？」 | する/しない選択 |
| 🚨 緊急 | 浴室30分動きなし、トイレ5分以上滞在等 | **要即対応** |
| 💇 髪洗い忘れ | お風呂後ドライヤー未使用 | 声かけ |
| 📝 家族の記録 | 「家族が〇〇を記録」 | 情報共有のみ |
| ☑️ 対応完了 | 「〇〇さんが対応しました」 | 情報共有のみ |

### 3-2. ボタンの意味

通知に下部表示される Quick Reply ボタン:

| ボタン | 意味 |
|---|---|
| ✓ 了解（対応します） | 内容理解、対応する |
| 対応不要（誤検知） | センサーの誤検知、再通知不要 |
| 食事じゃない | 食事候補通知の誤検知 |
| 祖母さん / 祖父さん など | 誰の行動か指定 |
| 他の家族 | 特定の人ではない |
| ロックする / しない | 炊飯器ロック確認時 |

### 3-3. 押し間違えた時

通常の通知（「了解/対応不要」）は **後から押し直せば上書き** されます。
ただし最初に押した家族の判断が記録されるので、間違えたら別のボタンを押し直してください。

### 3-4. 誰かが対応すると全員に共有

一人が対応すれば、家族全員のLINEに **「☑️ 〇〇さんが対応しました」** が流れます。
これで重複対応を防げます。

## 4. 主なコマンド

### 4-1. ステータス確認

| コマンド | 動作 |
|---|---|
| `状況` | 今日の食事/トイレ/お風呂のサマリ |
| `最後の食事` | 直近の食事時刻と経過時間 |
| `リンク` | 最新の家族管理画面URLが返信される |

### 4-2. 機器操作

| コマンド | 動作 |
|---|---|
| `ロック解除` | 炊飯器のロック解除（4桁コード送信→確認） |

### 4-3. 意見・質問

| コマンド | 動作 |
|---|---|
| `意見 内容` | 開発者へ意見送信 |
| `質問 内容` | 開発者へ質問送信 |
| `要望 内容` | 開発者へ要望送信 |
| `バグ 内容` | バグ報告 |

例:
```
意見 通知が多すぎて困ります
質問 タブレットのアプリ固定はどうやる？
バグ ロック解除ボタンが反応しません
```

### 4-4. その他

| コマンド | 動作 |
|---|---|
| `メニュー` | ボタンメニュー表示 |
| `ヘルプ` | コマンド一覧 |
| `登録 名前` | 自分を家族登録 |
| `登録解除` | 自分の登録を削除 |
| `登録一覧` | 登録済み家族を確認 |

## 5. よくあるシナリオ

### シナリオA: 食事候補の通知が来た

```
[システム] 🍱 12:30頃 「昼食」の動きをセンサーが検知しました
判定根拠: 炊飯器の蓋×1、冷蔵庫×3
誰の食事ですか？
[祖母さん] [祖父さん] [母さん] [他の家族] [食事じゃない]
```

**対応**:
- 祖母さんが食べたなら → [祖母さん]
- 祖父さんが家にいて食べたなら → [祖父さん]
- 自分が訪問して用意してあげたなら → [他の家族]
- 実際は何も食べてなかった → [食事じゃない]

### シナリオB: 炊飯器ロック確認

```
[システム] 🍚 祖母さんが本日3回目の食事を検知しました
前回の食事から 45 分経過
炊飯器をロックしますか？
[ロックする] [ロックしない]
```

**対応**:
- 食べすぎ防止したい → [ロックする]
- 実は1回目の食事だった（誤検知） → [ロックしない]

### シナリオC: トイレ長時間アラート

```
[システム] ⚠️ トイレに 6 分以上滞在しています
時刻: 14:30
祖母さんが転倒等で動けなくなっていないか、確認してあげてください。
[✓ 了解（対応します）] [対応不要（誤検知）]
```

**対応**:
- 即座に祖母さんに電話 → 応答あれば [✓ 了解]
- 応答なし → 至急近くの家族に連絡または訪問
- センサー誤検知が明らか → [対応不要]

### シナリオD: 家族管理画面を開きたい

1. メッセージで「メニュー」と送信
2. [🏠 家族管理画面] ボタンをタップ
3. ブラウザが開く → パスワード入力（家族グループの固定メッセージ参照）

URL が古くなった場合は「リンク」と送信すれば最新URLが返ります。

## 6. トラブルシューティング

### Q. 通知が来ない
- `登録 〇〇` で自己登録したか確認
- LINE ボットをブロックしていないか確認
- LINE グループ固定メッセージから友達追加し直す

### Q. ボタンが押せない / 反応しない
- LINEアプリを再起動
- 同じ通知に複数回ボタン押下しても、最後の判断が記録される

### Q. 「リンク」と送ってもURLが返らない
- 別の家族が試してみる（ボット側の問題かもしれない）
- 解決しない時は「バグ リンクが返りません」と送信

### Q. 知らない通知が来る
- メッセージをスクショして「質問 この通知の意味は？」と送信

### Q. 通知が多すぎる
- 「対応不要（誤検知）」を押し続けると、システムが学習して減少
- それでも多い時は「意見 通知が多すぎて困っています」と送信

## 8. 印刷について

このページの右下の **「🖨️ 印刷 / PDF」** ボタンをタップすると:
- ブラウザの印刷画面が開く
- 「PDFに保存」を選択 → ファイルとして保存可能
- そのまま印刷も可能（A4で2-3ページ程度）

家族で共有する印刷物として使えます。
