# 家族用ガイド ── IoT生活サポートシステム

おばあちゃんの生活をサポートするシステムの使い方ガイドです。

---

## アクセス方法

### 家族管理画面
- **自宅Wi-Fi接続時**: `http://taraberrypi.local:8000/family`
- **外出先**: LINEで届くURL + `/family`
- パスワードが必要です（お母さんに聞いてください）

### おばあちゃんのタブレット画面
- **自宅Wi-Fi接続時**: `http://taraberrypi.local:8000/tablet`
- **外出先**: LINEで届くトークン付きURL

---

## 家族管理画面の使い方

### ダッシュボード
画面上部に4つの数字が表示されます:
- **今日のイベント** ── センサーが検知したイベントの合計数
- **登録人数** ── システムに登録されている人数
- **消費電力 (W)** ── 炊飯器の現在の電力（リアルタイム更新）
- **祖母の食事回数** ── 今日の食事セッション数（3回以上で赤く警告）

### かんたん記録
おばあちゃんから報告を受けたら、ボタンを押して記録できます。

| ボタン | 用途 |
|--------|------|
| 起床 | 朝起きたことを記録 |
| お薬 | 薬を飲んだことを記録 |
| お風呂 | お風呂に入ったことを記録 |
| トイレ | トイレに行ったことを記録 |
| おやつ | おやつを食べたことを記録 |
| 就寝 | 寝たことを記録 |

- ボタンを押すと確認ダイアログが出ます（誤タップ防止）
- 記録するとタブレットのスタンプカードに即反映されます

### 日付ナビゲーション
- **「< 前日」「翌日 >」** ── 日付を切り替えて過去のイベントを閲覧
- **日付ピッカー** ── 特定の日を直接選択
- **「今日に戻る」** ── 今日のイベント表示に戻る

### イベント一覧
- **人物フィルタ** ── 「全員」「祖母」「母」「祖父」で絞り込み
- **人物変更** ── ドロップダウンで検知された人物を修正できます
- **時刻編集** ── 時刻をクリックすると編集できます
- **削除** ── 行の右端の × ボタンで削除（確認ダイアログあり）

### 炊飯器ロック
- 食事が多すぎるとシステムが炊飯器を自動ロック（電源OFF）します
- 画面上部に赤いバナーが表示され、「ロック解除」ボタンで手動解除できます

## LINE通知

以下の通知がLINEに届きます:

| 通知 | タイミング | 内容 |
|------|-----------|------|
| 食事検知 | 食事3回目以降 | 祖母の食事回数が多いことを通知 |
| 炊飯器ロック | 自動ロック時 | 炊飯器を自動ロックしたことを通知 |
| 浴室緊急 | 30分間無動 | 浴室で30分動きがないことを緊急通知 |
| お薬リマインド | 9時・12時 | お薬未服用の場合に通知 |
| お風呂リマインド | 18時 | お風呂未入浴の場合に通知 |
| 1日のまとめ | 22時 | スコア・達成/未達成の一覧 |
| システム異常 | 随時 | サーバがダウンした場合に通知 |

## おばあちゃんのタブレット画面

タブレット画面は**閲覧専用**です。おばあちゃんが操作する必要はありません。

表示される内容:
- **大きな時計** + 日付・曜日
- **時間帯ガイド** ── 「朝ごはんの時間」など今の活動を表示
- **炊飯量ガイド** ── 食事の時間帯に「ご飯は○合」と表示
- **注意喚起** ── 「さっき食べましたよ」「お薬まだですよ」
- **お花の成長** ── スタンプの達成数に応じて花が育つ（毎日色が変わる）
- **スタンプカード** ── 起床・お薬・朝食・昼食・お風呂・夕食・就寝
- **私のお庭** ── 過去14日分の花が庭に並ぶ成長記録

---

## 炊飯量の設定

食事の時間帯に表示される炊飯量は `.env` ファイルで変更できます:

```
RICE_GUIDE_朝食=1合
RICE_GUIDE_昼食=1合
RICE_GUIDE_夕食=2合
```

## トラブルシューティング

### 画面が表示されない
1. ラズパイの電源が入っているか確認
2. Wi-Fiに接続されているか確認
3. `http://taraberrypi.local:8000/` にアクセスしてみる

### LINE通知が来ない
- ラズパイがインターネットに接続されているか確認
- LINEの通知設定がONになっているか確認

### 外出先からアクセスできない
- LINEで届いた最新のURLを使ってください（再起動でURLが変わります）
- URLが届いていない場合はラズパイが再起動中の可能性があります

### おばあちゃんがタブレットを触ってしまった
- タブレット画面は閲覧専用なので、何を触っても問題ありません
- ホーム画面に戻ってしまった場合はアプリアイコンをタップしてください
