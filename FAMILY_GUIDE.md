## このシステムでできること（ざっくり）

おばあちゃんの家に置いた小さなコンピュータ（ラズパイ）が、
**炊飯器の電気** と **タブレット画面** をつないで動いています。

家族ができることは大きく分けて3つだけです：

| やりたいこと | 方法 |
|---|---|
| 様子を確認する | **LINEで「状況」と送る** |
| 何かを記録する／設定する | 家族管理画面（ブラウザ）を開く |
| おばあちゃんに伝言する | 家族管理画面から送る |

> 💡 普段は **LINEだけ** でほぼ完結します。細かい設定だけブラウザを使います。

---

## LINEでできる操作（一番よく使う）

LINE公式アカウントに **そのまま日本語で送るだけ** です。
コピペでOK。漢字・ひらがなどちらでも反応します。

### 📋 コマンド一覧

| 送る言葉 | 返ってくるもの |
|---|---|
| **状況** | 今日の食事回数・お薬・お風呂・トイレ・最後の食事のまとめ |
| **最後の食事** | 直前の食事と「○分前に食べた」 |
| **タスク** | 今日の家族タスク（誰が何をやる予定か）一覧 |
| **済 ○○** | タスクを「やった」と記録（例: `済 お薬確認`） |
| **ロック解除** | 炊飯器のロックを解除する手続きを始める |
| **リンク** | おばあちゃんのタブレットを外から見るURL |
| **ヘルプ** | このコマンド一覧をLINEで返信 |

### 🔓 炊飯器のロック解除のやり方（重要）

おばあちゃんが何度も炊飯しようとしたとき、システムが自動で炊飯器の電源を切ります。
このときLINEに通知が来ます。**家族が必要と判断したら**以下の手順で解除します：

1. LINEに **「ロック解除」** と送る
2. 4桁の数字（PIN）が返ってくる
3. その数字を **そのままLINEで返信** する
4. 「解除しました」と返ってくれば成功

> ⚠️ PINは **5分間だけ有効**。3回間違えると **15分ロック** されます。
> 急いで間違えるよりも、落ち着いて入力してください。

### 💡 こんな時どうする？

- **外出先からタブレットを確認したい** → 「リンク」と送る → 届いたURLをタップ
- **今日もう食べたか不安** → 「最後の食事」と送る
- **担当のタスクをやり終えた** → `済 朝のお薬確認` のように送る

## LINEに自動で届く通知

家族側から何もしなくても、こちらが必要なときだけ通知が来ます。

| 通知 | いつ | 意味 |
|---|---|---|
| 🍚 食事検知（多すぎ） | 食事が3回目以降を検知したとき | 「また食べようとしている」サイン |
| 🔒 炊飯器を自動ロック | システムが炊飯器の電源を切ったとき | 食事しすぎ防止のため自動で切った |
| 🚿 浴室で30分動きなし | 入浴中に動きがなくなったとき | **緊急**：すぐに様子を見に行く |
| 💊 お薬リマインド | 9時・12時 | お薬を飲んだ記録がない場合 |
| 🛁 お風呂リマインド | 18時 | お風呂に入った記録がない場合 |
| 📋 1日のまとめ | 毎晩22時 | スタンプ達成・食事回数・花の状況 |
| 📊 週次レポート | 毎週日曜22時 | 1週間の食事傾向・気になる傾向 |
| 🌐 公開URL更新 | ラズパイ再起動時 | 外出先用のURLが新しくなった |
| ⚠️ システム異常 | サーバが落ちた等 | ラズパイに何か問題が起きている |

## 家族管理画面（ブラウザ）の使い方

### 開き方

| 場所 | アドレス |
|---|---|
| **おばあちゃんの家のWi-Fiにいる時** | `http://taraberrypi.local:8000/family` |
| **外出先** | LINEで届く最新URLの末尾に `/family` をつける |

ログイン画面が出るので、**家族用パスワード**（家族グループに共有済み）を入力。

### 画面の見方

開くとこんな順に並んでいます：

#### ① ダッシュボード（一番上の数字）
- 今日のイベント数 / 登録人数 / 炊飯器の電力 / 今日の食事回数
- **食事回数が3回以上で赤くなる** → 注意のサイン

#### ② 機器管理（炊飯器のロック状態）
- 🔓 通常 / 🔒 ロック中 が常に見える
- ボタンで **手動ロック** や **手動解除** ができる
- LINEからの解除と同じ機能（パスワードログイン済みなので即効）

#### ③ かんたん記録（家族が見たことを記録）
おばあちゃんに会って様子を見たら、ボタン1つで記録できます：

| ボタン | いつ押す |
|---|---|
| 起床 | 朝起きたのを見たとき |
| お薬 | 飲んだのを見届けたとき |
| お風呂 | 入ったのを確認したとき |
| トイレ | 行ったのを確認したとき |
| おやつ | 食べたのを見たとき |
| 就寝 | 寝たのを見届けたとき |

> 押すと **タブレットのスタンプに即反映** されます。
> 確認ダイアログが出るので誤タップの心配なし。

#### ④ 7日間ヒートマップ
過去7日 × 24時間で、おばあちゃんがいつ動いていたかを色の濃さで表示。
**生活リズムの異変** を視覚的に発見できます。

#### ⑤ 家族タスク
「朝のお薬確認は母」「夕食の確認は孫」のように **担当・時刻** を登録すると、
時間になると **担当者のLINEに通知** が来ます。

#### ⑥ お薬スケジュール
朝・昼・夜のお薬の **時刻** を設定。タブレットのリマインドや、未服用通知の判定に使われます。

#### ⑦ 炊飯量ガイド
「次に何合炊くか」を1合単位で指定。
食事の時間帯に **タブレットに「ご飯は○合 炊いてね」** と表示されます。
クリアすれば非表示にもできます。

#### ⑧ タブレット伝言
おばあちゃんへのメッセージを送る機能。プリセット4種（お薬／お風呂／ご飯／電話）または自由入力。
**60分でタブレット側から自動で消える** ので、古い情報が残りません。

#### ⑨ イベント一覧
- センサーが検知した出来事や記録が時刻順に並ぶ
- **人物フィルタ**：「祖母」だけに絞り込み等が可能
- **人物変更**：センサーが間違えたら正しい人に修正できる
- **時刻クリック → 編集**、**× ボタン → 削除**
- **日付ナビ**：「< 前日 / 翌日 >」「日付指定」「今日に戻る」で過去日も見られる

## おばあちゃん側の動き（家族の理解用）

ここは **「おばあちゃん側からはどう見えているか」** を家族が理解するためのセクションです。
おばあちゃんに直接見せるものではありません。

### 🌸 タブレットに何が映っているか

リビングのタブレットには **「自分の記録帳」** として表示されます。
監視カメラとは思っていません。**「お花が育つ手帳」** だと思っています。

タブレットに映るもの：

1. **大きな時計と日付** — いつでも今の時間がわかる
2. **時間帯の挨拶** — 朝なら「おはようございます」、夜なら「お疲れさまです」
3. **やさしい注意喚起**（直接的な指摘はしません）
   - さっき食べた直後 → 「さっき たべましたよ」
   - 食事が3回以上 → 「きょうは よく たべましたね」
   - 9時以降にお薬未確認 → 「おくすり のみましたか？」（点滅）
4. **お花の成長** — 起床・お薬・食事・お風呂などの記録が増えると花が育つ（8段階）
5. **わたしのおにわ** — 過去14日分のお花が並ぶ庭
6. **スタンプカード** — 今日の達成項目（起床・お薬・朝食・昼食・お風呂・夕食・就寝）
7. **家族からの伝言** — 青いカード。「わかった」を押すと消える
8. **音声読み上げ** — 通知やメッセージを声で読み上げ（スピーカー🔊ボタンでON/OFF）
9. **炊飯量** — 家族が登録していれば「ご飯は2合 炊いてね」と表示

### 💡 おばあちゃんが押せる唯一のボタン：「できた」

スタンプカードの未達成項目に「**できた**」ボタンが表示されます。
押すと、システムが **裏でセンサー記録と照合** します：

- センサー記録あり → スタンプOK、家族のLINEに「確認済み」と通知
- センサー記録なし → スタンプ付かず、おばあちゃんには「家族に伝えてください」と表示。家族のLINEには「確認してください」と通知

> このため **家族が知らないところで適当にスタンプが付くことはありません**。
> 30分間は連打防止のクールダウンが入ります。

### 🍚 食事しすぎを防ぐしくみ

おばあちゃんから見える流れ：

1. 1回目の食事 → 普通に炊ける
2. 90分以内に2回目の食事行動 → タブレットに「さっき たべましたよ」
3. それでも炊こうとする → **炊飯器の電源が入らない**（家族には通知）
4. 「機械の調子が悪いみたい」とおばあちゃんは思う
5. 家族が必要と判断したらLINEから解除（→ おばあちゃんはこの操作に気づかない）

### 🚿 お風呂の安全監視

浴室ドアと脱衣所に小さなセンサーがあります（おばあちゃんには見えない位置）。

1. 浴室に入る → 自動で「お風呂中」を記録
2. 30分間動きがない → **家族のLINEに緊急通知**
3. 浴室から出る → 自動で「お風呂完了」

> 過去に浴室で意識を失ったことがあるため、安全のための重要な機能です。

### 🤐 おばあちゃんに絶対知られないこと

家族の操作は **おばあちゃん側のタブレットには一切表示されません**：

- 家族がイベントを編集・削除した
- 家族が「かんたん記録」でスタンプを付けた
- 家族が炊飯器をロック／解除した
- 家族同士のLINEのやり取り

**「機械が勝手に記録している」** と思ってもらうのが基本設計です。

## 困ったとき

### LINEから返信が来ない
1. しばらく待つ（10秒くらい）
2. 「ヘルプ」と送ってみる
3. それでも来ないなら **ラズパイの再起動中** の可能性 → 数分待つ
4. それでもダメなら家族の中の詳しい人（孫）に連絡

### タブレットが真っ黒／違う画面になっている
- 画面をタップして起こす
- ホーム画面に戻ってしまったら **アプリアイコン**（記録帳）をタップ
- それでもダメなら **タブレットの再起動**（電源ボタン長押し）

### 家族管理画面が開かない
1. Wi-Fi に繋がっているか確認
2. 外出先なら **LINEで「リンク」** と送って最新URLを取得 → 末尾に `/family`
3. パスワードを再確認

### 外出先から繋がらない
- ラズパイが再起動するとURLが変わります
- LINEで「**リンク**」と送れば最新URLが返ってきます

### おばあちゃんがタブレットを変なところ触ってしまった
- 心配いりません。**閲覧専用なので壊れません**
- ホーム画面に戻ってしまっただけならアプリアイコンをタップ

### 通知が来すぎる／来ない
- 家族管理画面の **詳細設定** から ON/OFF やしきい値を調整できます

## 絶対に守ること

1. **おばあちゃんに「監視している」と絶対言わない**
   - タブレットは「記録帳」と説明してあります
   - カメラの存在も伏せてあります

2. **「また食べたでしょ」と直接指摘しない**
   - タブレットが「さっき食べましたよ」とやさしく伝えます
   - 家族は「そうみたいだね、お花育ったね」のように記録に乗っかる

3. **家族の操作はおばあちゃんの前でやらない**
   - ロック解除・編集・伝言送信などは別室で
   - おばあちゃんは **「機械の調子が悪い」** と思っています

4. **ロック解除は慎重に**
   - 「食べたがっているから」だけで解除しない
   - **本当に必要なときだけ**（家族が一緒に食事するなど）

5. **緊急通知（浴室30分無動）は最優先**
   - 来たら **すぐに様子を見に行くか電話する**

## まとめ：1日のおすすめ運用

| 時間 | やること |
|---|---|
| 朝 | LINEに「状況」を送って様子確認 |
| 昼 | 通知が来たら対応、なければ気にしない |
| 夕方 | お風呂リマインド通知が来たら気にする |
| 夜22時 | 「1日のまとめ」が届く → ざっと目を通す |
| 週日曜22時 | 「週次レポート」が届く → 傾向をチェック |

> **「気になったらLINEで状況」** これだけ覚えていれば大丈夫です。
